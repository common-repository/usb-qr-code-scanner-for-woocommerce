== Description ==

A plugin for quick ordering Payment on the spot using by barcode reader(USB Qr Code Scanner)

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. create  page and use shortcode [usbqrcode] in page
4. Create User with role [seller or shop_manager or administrator]
5. login to your account 
6. Connect USB QR Code Scanner To Your Computer and Scan the code on the page created at step 3
